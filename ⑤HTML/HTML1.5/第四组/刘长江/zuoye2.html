 本章重点:

sqlserver数据类型:
分为二进制数据类型、字符数据类型、日期类型、数字数据类型、布尔型。

二进制数据类型:image 用来存储图像

字符数据类型:
a)char：固定长度的非unicode数据
b)varchar:长度可变的非unicode数据
c)nchar:固定长度的unicode数据
d)nvarchar:可变长度的unicode数据


日期类型:dateime

数字数据类型:
a)int：整数
b)smallint:短整数
c)float:小数

布尔型:
Bit:只有两个值0，1 0：假 1：真

gbk：一般国内使用
utf-8：国际化

常见的编码:
unicode:unifying code 统一码
ASC码表 ASCII（美国信息交换标准代码）
iso-8859-1：又称Latin-1或“西欧语言”
gbk：GBK全称《汉字内码扩展规范》（GBK即“国际”、“扩展”汉语拼音的第一个字母，英文名称：Chinese lntemal Code Specification）
UTF-8:UTF-8(8-bit Unicode Transformation Format)是一种针对Unicode的可变长度字符编码，又称万国码
